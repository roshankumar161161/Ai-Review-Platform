# AI Review Platform

An AI-powered web application that analyzes  product reviews using AI.

#Website : https://satisfied-amazement-production-7bba.up.railway.app/dashboard

## Features

- Analyze YouTube video reviews
- Product review analysis
- AI-powered sentiment analysis
- Fake review detection
- Modern Next.js frontend
- FastAPI backend
- Railway deployment

## Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Python

### AI & APIs
- Google Gemini API
- YouTube Data API v3

## Installation

```bash
git clone https://github.com/USERNAME/Ai-Review-Platform.git
```

Install frontend

```bash
cd frontend
npm install
npm run dev
```

Install backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```
